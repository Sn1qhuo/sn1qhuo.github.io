裁员还不够，GitHub 要求全员转 Teams 惹开发者吐槽！(转载)

本文为转载自——CSDN博主「CSND资讯」，文末附文章原链接

整理 | 邓晓娟 责编 | 梦依丹

出品 | CSDN（ID：CSDNnews）

近日，全球知名开源代码托管平台公司正在经历新的一轮裁员潮。

2 月初，微软旗下的 GitHub 宣布将在本财年结束前裁员10%。目前 GitHub 拥有大约3,000名员工，也就是说，在财年结束前将会有约300名员工被解雇。

裁员的原因很简单：预算调整。

紧接着，GitHub 还宣布等办公室租期一结束，就关闭所有办公室。GitHub 首席执行官 Thomas Dohmke 表示，“我们发现，在世界各地的办公室利用率都非常低，该决定证明了我们长期存在的远程优先文化的成功。”

还提出了其他削减成本的方法，包括全员转为远程办公，以提高运营效率；所有员工的电脑更换频率，从之前的 3 年一换增加到 4 年一换；并且视频会议工具从 Slack 切换为微软 Teams，以大量节省成本并简化跨公司和客户的对话，而 Slack 将作为日常协作工具使用。

Github公告内容

对于 GitHub 突如其来的一刀切裁员方式，许多网友和业界人员发表了自己的想法。有人认为 GitHub 的裁员原因和做法将成为一种趋势，因为如今许多公司都在“工具”上投入了许多成本，却没有创造出理想中的效益。

也有持不同意见的业内人士，认为裁员和远程办公会加速真正的人才流失。

Linear 创始人/首席执行官 Karri Saarinen 认为，裁员的行为可以理解为缩减开支，但关闭办公室、并要求更换工具、降低工具维护成本这样的举措，是一个企业释放出来的危险信号。

因为在这种小事情上，自上而下的决定是企业变革的信号。

Karri Saarinen 表示，这可能是许多企业的常态，财务人员或技术人员在做这些决定时，只看到了“成本”。他引用了美国硅谷著名企业家 Steve Blanc 的说法，“……企业通常为了节省一万美元左右的成本，无意中造成最好的工程师大批离职的局面……随着初创公司逐渐壮大，创始人和董事会需要意识到，最重要的转型并不在于系统或硬件，而是公司最宝贵的资产——员工。”

另一方面，Karri Saarinen 对于 GitHub 将要求员工们转向使用 Teams 感到不认同。他认为，公司的领导层经常忽略合适的工具的重要性。在远程办公过程中，员工每天使用的工具就像是公司的办公室一样，是相当于“环境”的存在。如果这种环境导致了挫折感，那这种挫折感就会渗透到其他方面。

这也是造成裁员以外另一个争议的源头。GitHub 要求员工们使用 Teams 的举措，引发了许多网友的讨论。在 HN 论坛上，许多人提出了关于自己使用 Teams 的感受，大多数评论都认为 Teams 使用体验感一般，经常“停止响应”和出现“随机错误”。

HN论坛正在讨论Teams的使用感受

许多人认为 Teams 想要集大成，却连基本的功能正常运行都无法保证。非常影响体验感。Karri Saarinen 对于这个问题更是作出尖锐的评论：“当微软领导层觉得 GitHub 的裁员还不够糟糕时，不得不通过强迫大家在使用了 4 年的笔记本电脑上使用 MS Teams 来增加痛苦。”

对于裁员和远程办公、更换协作工具的举措，GitHub CEO Thomas Dohmke解释道，这一切都是为了让公司能有更长期的稳定增长。并在 2023 年 9 月 1 日前统一切换到 Teams，在此之前仍然使用 Slack 作为日常协作工具。

同时表示，GitHub 未来将大力关注人工智能，他希望 GitHub 成为”未来世界的开发者优先工程系统”。

从高中到大学 寻找真实的自己(转载)

本文为转载自——CSDN博主「一只倔强的小bug」，文末附文章原链接

写在前面

这是这个寒假刚开始在CSDN上写博客的时候发的第一个blink，当时想说的话有点多，但blink的文字限制是1024字，所以那时控制了字数，现在放开重新写。

写在正文

因为疫情原因在家上了差不多3个月的网课，大一回来过个寒假，再次回到学校已经快成了大二，网课生涯，从寒晚冬到炎初夏，本来很幸运选上的羽毛球课上成了体育与健康讨论课，每节课换一个讨论话题，大学物理实验做成虚拟实验（实在不理解为什么计算机与软件学院的学生要学大学物理和大学物理实验实验这两门课），平常上课摸鱼和划水变得简单，就像是完成任务一样，不开摄像头的日子，变换着地点和姿势听着网课，Ctrl+C变得更加重要，不得不让人想起高中别样的生活。

中学生涯经常思考这样一个问题，我到底在干些什么？我想要什么？我在追求什么？我所努力奋斗的目标是什么？为什么我需要考上一所好大学？每个人的成长经历不一样，每个人的想法也不一样。就我而言，我的梦想就是，能在将来的某个时候帮助到很多人，为很多人提供一个成长的平台，给予有需要的人服务。这意味着我需要一个好的想法，这个想法能够符合时代发展的要求，开历史倒车是活不久的，也就是说我必须具备敏锐的观察力，能够发现可行的办法，那么我需要接受好的教育。其次我还需要资本和人脉，没有资本首先连自己的养不活，没有人脉意味着单打独斗也很难搞到资本，这显然是不行的。要资本我必须得有很好的工作或者很棒的想法（棒到足够吸引资本），这意味着我必须具备优秀的技能和足够的知识，那么我需要接受良好的教育。要积累人脉意味着我我社交要好，首先就是得有一个好的平台，给我结识的机会，其次于我本身而言，我必须文化涵养要好，待人接物要好，礼节要到位，那么我需要接受良好的教育。

这就是我当时的想法，就是想考上好大学，去接受好的教育，凭借这种信念，于是努力学习。

两年前的2020年，那时高二，同样是在家上网课，那时老老实实，勤勤恳恳，认真上好每一节网课。还记得去年高三寒假天天早起读书背单词，很多人说高中是自己知识的巅峰，现在想想，好像是那么回事。

但是高三的时候觉得真的很无聊，感觉每天都是干一样的事情，在高中老师的积极宣传下，那时候总是想象大学有多美好，未来有多光明，每天只想着早点高考，早点结束这乏味令人窒息的清道夫式生活。

在高考那几天，老实说，没什么感觉，生活总是匆匆忙忙，高考也一样，让人还来不及回味，突然间它就飘走了。 这突然间的解放，总是让人不知所措。但是太久的束缚，让人太想疯狂一把，流连在声光色影之中。

高考成绩出来的时候，还是感叹了一下，原来真的不是所有人都能考上清华北大，习惯性伤心几天，但也没有影响到放飞自我的心态。然后玩了三个月，来了深大学计算机之后，才发现，原来大学并不是没有一点点人间疾苦的人间天堂，而是冷暖自知的真实社会，这里的老师很坦诚很淡定很平静地讲述着这个社会的现实，愉快地看着我们脸上由一开始的兴奋到茫然失措的复杂变化表情。 但是人总是有惰性的，高考前一百天的誓师大会让人热血沸腾，但结果表明，最后发光的往往是那些一脸平静的学生。打鸡血会给人一腔热血，但决定人的行为的还是人本身。这一学期下来，这群大一新生大多是这样的：早上如果没有早课，那是昨晚就要计划好的事情，你问几点起床，答案是：他们是不会起床的，他们可能昨晚就计划好今天中午点什么外卖作为早餐或者是晚餐。有早课其实也影响不大，看是什么老师，不点名就不去了，点名的话，现在大学生创业精神极其可嘉，代孕虽然还没有，但找人代课还是比较抢手的买卖。实在没有也没关系，经过长期的专业训练和摸爬滚打，已经形成组织化团体化行动，一般以宿舍为单位，舍友彼此签订协议，每次行动派一名出动打听消息，这次你替我，下次我救你，点名的时候你是我，我是你，为了不引起老师怀疑，必须学习专业狙击手，严格遵守打一枪换一位，报个到换个位的准则。 我宿舍还好，都是玩到晚上十二点就睡了的，其他宿舍的话，早上起来，可以看到有的宿舍亮着灯，在排除了早起学习的可能性之后，那一般是昨晚忘记关了。

但有的人一直秉承着宁可累死自己，也要卷死同学的传统观念，经常是，不在图书馆，就在去图书馆的路上。 这类同学经常会受到其他同学的同时存在的仰望和鄙视，亲身经历过。经过仔细的研究加实践表明，内卷必须采取地下行动，地点一定要足够隐蔽，避免被熟人认出，像图书馆和实验室是良好的作案地点。如果你想在宿舍学习，根据打不过就加入的最优解法，要么你感化舍友（几率可忽略不计），要么让舍友在某个不传统的领域带飞你。

写在后面

我的一位老师经常向我们班的学生灌输他的想法说：本科，只是一个大众教育而已，不要轻易离开学校，离开学校的那一天，很有可能是你人生的学习至高点，要做有困难的事情，干有行业壁垒的工作，要到全世界最好的大学去读研读博，到全世界最好的实验室去打怪升级，成为顶尖的人。
